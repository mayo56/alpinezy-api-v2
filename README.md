# Alpinezy API v2

## Configuration
### Installer les différents module
**Requière NodeJS**
```sh
npm i
```
### fichier .env dans le dossier config
```md
# Token secret pour les token de connexion
TOKEN_JWT="token"

# Accès db
DB_USER="user"
DB_URL="url"
DB_PASSWORD="password"
```

# Démarrer l'api
Très simple !
```sh
npm run start
```
