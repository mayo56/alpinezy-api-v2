# Alpinezy API v2

## Configuration
### Installer les différents module
**Requière NodeJS**
```sh
npm i
```
### fichier .env dans le dossier config
"**/config/.env**"
```md
# Token secret pour les token de connexion
TOKEN_JWT="token"

# Accès db
DB_USER="User"
DB_HOST="url"
DB_DATABASE="nom de base de donnée"
DB_PASSWORD="mot de passe"
```

# Démarrer l'api
Très simple !
```sh
npm run start
```
